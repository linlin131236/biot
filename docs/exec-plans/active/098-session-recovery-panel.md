# M98 Exec Plan - 会话恢复体验

## 目标
把会话恢复状态做成中文只读面板，说明能恢复什么、不能恢复什么、恢复前还需要哪些安全检查。

## 参考资料
- Agent产品化流水线：恢复必须先解释边界，再等待人工动作。
- 桌面AI编程Agent全流程架构对比：恢复不能绕过权限复查。

## 实施
- 新增 `SessionRecoveryPanel.tsx` 和测试。
- 新增 `session_recovery_api.py`，输出恢复状态、快照说明和安全提醒。
- 不提供自动 resume 按钮。

## 验收
- 中文显示恢复状态。
- 不自动恢复、不跳过 PermissionGate。
- 测试覆盖只读安全文案。
