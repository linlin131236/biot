# Design

Bolt should feel like a quiet workbench, not a marketing page.

## Product Surface

- Show the current task, current workspace, Agent Core status, pending permissions, and trace events.
- Keep destructive actions explicit and reviewable.
- Prefer dense, readable controls over decorative cards.

## Interaction Rule

The user must always be able to answer: what is Bolt doing, what does it want to change, and what happens if I approve?
