# M144 Decision - 设置中心产品化

## 决策

采用 `LiquidGlassSettingsSurface` 承载设置中心的分类内容，让左侧分类导航真正切换不同配置页面。

## 背景

M141-M143 已经完成液态玻璃 shell、组件 primitives 和任务驾驶舱，但设置中心仍偏骨架，点击不同分类后缺少差异化内容。

## 结果

- 设置中心按分类显示常规、代码预览、模型设置内容。
- API 密钥只展示配置状态和安全边界，不回显明文。
- 新增内容仍是前端只读展示，不触发工具执行。

## 风险

- 当前只覆盖前三个设置分类，后续 M145-M149 继续补权限、补丁、审计、验证、记忆和团队页。
