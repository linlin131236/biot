# Bolt V1 Product Spec

Bolt V1 is a safe desktop coding agent.

## Must Have

- Configure an OpenAI-compatible model endpoint.
- Select a local workspace.
- Read and search workspace files through tool requests.
- Propose file changes through change sets.
- Require confirmation before writes and shell commands.
- Record trace events for every tool request.
- Promote unresolved failures into P0 context.

## Not V1

- Autonomous PR creation.
- Browser automation.
- Multi-agent workforce.
- Plugin marketplace.
- Cloud execution.
